# Tower Progression Prototype

A small Godot game project built as a learning exercise, with the long-term goal of evolving it into a simple, fun game I can gift to my nephew.

This project is intentionally starting small. The first goal is not to build a massive RPG, but to create one complete playable loop: leave town, enter the tower, defeat an enemy, collect materials, craft a weapon, and complete the demo.

## Project Goal

The main purpose of this project is to help me learn game development by building a real, playable game piece by piece.

I want to practice turning a rough game idea into something structured, playable, and finishable. Instead of trying to build the full dream game immediately, this project begins as a tiny prototype and will grow through small, manageable versions.

Eventually, I would like this to become a personal game made as a gift for my nephew.

## Core Game Idea

The game is a small tower-climbing RPG where the player progresses through floors, gathers materials, defeats monsters, crafts equipment, and reaches safe hub areas.

The basic loop is:

```text
Start in Town
→ Enter the Tower
→ Fight an Enemy
→ Collect Loot
→ Return to Town
→ Craft Gear
→ Progress Further
```

# Tower Progression Slice v0.0.1 PRE-RELEASE

A small playable Godot prototype demonstrating a complete tower progression loop.

## Current Gameplay Loop

- Start at the title screen
- Enter the hub
- Explore Floor 1
- Fight a goblin
- Forage materials
- Return to the hub
- Craft a D-Rank Dagger
- Receive the demo completion message


## Current Limitations

- Combat is currently a placeholder interaction
- No save/load system
- No real inventory UI yet
- No production art or audio yet
- Not production ready
